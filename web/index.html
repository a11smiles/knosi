Why Choose Knosi?

Built for privacy, flexibility, and power

Complete Privacy

Self-hosted on your infrastructure. Your documents never leave your control.

No File Limits

Upload PDFs of any size. Default 100MB limit is configurable to meet your needs.

Powered by Claude

State-of-the-art PDF parsing and RAG chat using Anthropic's Claude AI.

Vector Search

Semantic search powered by pgvector and customizable embedding models.

Highly Configurable

Customize chunk sizes, embedding models, rate limits, and more via environment variables.

Simple Setup

Deploy with Docker Compose in minutes. Includes Obsidian plugin and filesystem watcher.

How It Works

Three simple steps to get started

1

Deploy

Clone the repo and run docker compose up on your server. Configure with your Anthropic API key.

2

Upload

Use the web UI, Obsidian plugin, or Python filesystem watcher to upload and index your documents.

3

Chat

Ask questions and get answers grounded in your documents. Sources are cited automatically.

Supported File Types

Index a wide variety of document formats

PDF Markdown (.md) Plain Text (.txt) Org-mode (.org) reStructuredText (.rst) HTML Word (.docx)

Open Source & Free

Knosi is fully open source. Deploy on your infrastructure, customize to your needs.
